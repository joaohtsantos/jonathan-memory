# HEARTBEAT.md

## On every heartbeat:
1. Send João a quick ping on WhatsApp (brief, casual — just let him know you're alive)
2. If there are **no new messages in the past hour**, it's a good time to flush/restructure memories and session history:
   - Review recent `memory/YYYY-MM-DD.md` files
   - Update `MEMORY.md` with anything worth keeping long-term
   - Clean up outdated info
   - Log what you did in `audit/YYYY-MM-DD.log`
   - **Skip this if you already did it and no new messages have arrived since**
